# 更新日志

## [1.4.0] - 2026-01-26

### 新增
- 🎵 新增播放功能：回复序号即可播放对应歌曲
- 🔗 自动生成 Neko云音乐播放链接
- 💡 添加使用提示，引导用户如何播放音乐
- 📋 搜索结果自动缓存，支持会话内播放

### 优化
- 🎯 优化用户交互体验，简化播放操作
- 💫 提升消息回复的流畅性

## [1.3.0] - 2026-01-26

### 新增
- ✨ 使用 PIL 生成精美的搜索结果图片
- 🎨 添加渐变背景、圆角卡片等现代设计元素
- 📝 显示所有搜索结果(不再限制为 5 首)
- 🖼️ 自动获取并显示每首歌的封面图片
- 📋 底部添加版权信息(两行显示)

### 优化
- 🔧 使用本地字体文件解决中文乱码问题
- 📏 调整封面尺寸为 100x100,确保不超出容器
- 🎯 优化图片布局和间距
- 💫 提升整体视觉效果

### 修复
- 🐛 修复 Linux 系统上中文显示乱码的问题
- 🐛 修复封面超出卡片边界的问题

## [1.2.0] - 2026-01-26

### 新增
- ✨ 使用合并转发消息展示搜索结果
- 📦 每首歌独立显示在转发消息中
- 🎨 优化歌曲信息展示格式,添加表情符号

### 优化
- 🔄 改进消息发送方式,使用 forward_result
- 💫 提升用户体验,搜索结果更清晰

## [1.1.0] - 2026-01-26

### 新增
- ✨ 添加音乐封面显示功能
- 🖼️ 每首歌的封面和对应信息一起显示
- 📝 优化搜索结果的展示格式

### 优化
- 🎨 改进消息链构建,封面和信息不再分离
- 🔧 优化代码结构,提高可维护性

## [1.0.0] - 2026-01-26

### 新增
- 🎉 首次发布
- ✨ 支持通过 `点歌 歌曲名` 搜索音乐
- 🔍 集成 Neko云音乐 API
- 📋 显示歌曲详细信息(名称、歌手、专辑、ID)
- 🚫 无需前缀,直接触发指令

---

**版本说明**: 版本号格式为 `主版本号.次版本号.修订号`
- 主版本号: 重大更新或不兼容的修改
- 次版本号: 新增功能或重要改进
- 修订号: Bug 修复或小的改进
