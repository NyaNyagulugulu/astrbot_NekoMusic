# 更新日志

## [1.2.0] - 2026-01-26

### 新增
- ✨ 使用合并转发消息展示搜索结果
- 📦 每首歌独立显示在转发消息中
- 🎨 优化歌曲信息展示格式,添加表情符号

### 优化
- 🔄 改进消息发送方式,使用 forward_result
- 💫 提升用户体验,搜索结果更清晰

## [1.1.0] - 2026-01-26

### 新增
- ✨ 添加音乐封面显示功能
- 🖼️ 每首歌的封面和对应信息一起显示
- 📝 优化搜索结果的展示格式

### 优化
- 🎨 改进消息链构建,封面和信息不再分离
- 🔧 优化代码结构,提高可维护性

## [1.0.0] - 2026-01-26

### 新增
- 🎉 首次发布
- ✨ 支持通过 `点歌 歌曲名` 搜索音乐
- 🔍 集成 Neko云音乐 API
- 📋 显示歌曲详细信息(名称、歌手、专辑、ID)
- 🚫 无需前缀,直接触发指令

---

**版本说明**: 版本号格式为 `主版本号.次版本号.修订号`
- 主版本号: 重大更新或不兼容的修改
- 次版本号: 新增功能或重要改进
- 修订号: Bug 修复或小的改进
